# ILs Sheet Rules
*These rules and guidelines apply equally to the spreadsheet tabs other than **ILs**, but aren't currently rigidly enforced.*

1. All **times must be recorded directly from the in-game timer** provided by the **Shine Get Timer** practice code, version 2 or later (from Oct 2019), *except* for any times that are retimed (from e.g. full-game runs or with the fast text code accidentally left off).
